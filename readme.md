## 📖 Sobre o Projeto

O **Gerador de Tabuada Dinâmica** é uma aplicação desenvolvida com HTML, CSS e PHP que permite gerar tabuadas automaticamente a partir de um número escolhido pelo usuário.

Ao informar um número, o sistema calcula e exibe dinamicamente toda a tabuada correspondente de forma prática e organizada.

Exemplo:

Se o usuário escolher o número:

```txt
9
```

A aplicação exibirá:

```txt
9 x 1 = 9
9 x 2 = 18
9 x 3 = 27
...
```

---

## 🚀 Funcionalidades

- Entrada dinâmica de números
- Geração automática da tabuada
- Exibição organizada dos resultados
- Interface simples e intuitiva
- Processamento dinâmico com PHP
